# Conexão Psi

Rede de encaminhamento entre psis com foco em visibilidade profissional e conexão entre demanda e disponibilidade clínica.

---

## 🌱 Sobre o projeto

A Conexão Psi é uma rede criada para facilitar o fluxo de encaminhamentos entre profissionais da psicologia.

Muitos psis enfrentam dois cenários ao mesmo tempo:
- agenda cheia e pacientes sem vaga
- agenda disponível sem demanda suficiente

Este projeto surge como uma solução simples:
**conectar psis entre si de forma ética, organizada e prática.**

---

## 🧠 Como funciona

Ao entrar na rede:

- Seu perfil é adicionado ao diretório de psis
- Outros profissionais podem visualizar seu perfil
- Você pode receber encaminhamentos dentro da comunidade
- Sua visibilidade aumenta dentro da rede ativa

O sistema é simples e direto, sem burocracia.

---

## ⭐ Plano Premium (opcional)

O plano premium não é obrigatório.

Ele serve apenas para aumentar visibilidade dentro do diretório:

- Destaque no topo
- Bio profissional ampliada
- Exibição de valor de sessão
- Botão direto de WhatsApp

---

## ⚙️ Ativação

Após confirmação do pagamento:

- o perfil é ativado automaticamente
- o profissional já aparece no diretório
- não é necessário envio manual de dados adicionais

---

## 🧭 Ética

A Conexão Psi respeita integralmente as diretrizes éticas da profissão, incluindo normas do CRP.

O projeto não substitui supervisão clínica, nem interfere na autonomia profissional.
